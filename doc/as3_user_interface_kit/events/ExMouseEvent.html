Classpublic class ExMouseEvent
InheritanceExMouseEvent flash.events.MouseEvent

ExMouseEventクラスは、MouseEventに関連するイベントを定義します。次のようなイベントがあります。
  • MOUSE_DOWN_LONG_TIME:表示オブジェクト上でMouseDownしたまま一定の時間が経過した後、連続的に送出されます。
