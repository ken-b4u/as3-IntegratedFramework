Packageas3_animation_kit
Classpublic class Animator
InheritanceAnimator flash.events.EventDispatcher

AnimatorクラスはNumericalInterpolatorをTweenerライクに使用するためのラッパークラスです.

NumericalInterpolatorを使用したほうが高速な動作が期待できますが、 より手軽にアニメーションさせたい場合は Animator クラスを使用します。
