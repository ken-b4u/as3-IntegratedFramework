Interfacepublic interface ISceneProvider
Implementors SceneManager

主にDeepLinkNavigationControllerのデータソースとなる為に実装が必要なインターフェースです。 DeepLinkNavigationControllerは、SWFAddressEvent.CHANGEを監視していて、 イベント発生時にSWFAddressEvent.getPathNames()の最後に入っているパラメータをシーンIDとして、 ISceneProvider.getSceneByIDを呼び出します。 この時ISceneProvider.getSceneByIDを実装したクラスで適切なIViewControllerを返せば、 シーン管理の手間が大幅に節約できます。
