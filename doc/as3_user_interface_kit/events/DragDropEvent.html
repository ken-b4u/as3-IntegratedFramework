Classpublic class DragDropEvent
InheritanceDragDropEvent flash.events.Event

DragDropEventクラスは、表示オブジェクトのDrag&Dropに関連するイベントを定義します。次のようなイベントがあります。
  • CHANGE_RECEPTION:Drag可能オブジェクトの受け入れ先が変更された時に送出されます。
  • START_DRAG:オブジェクトがDrag開始された時に送出されます。
  • STOP_DRAG:オブジェクトのDragが終了した時に送出されます。
