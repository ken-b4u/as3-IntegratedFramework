Classpublic class ButtonController
InheritanceButtonController ViewController flash.events.EventDispatcher
Subclasses ToggleButtonController

Button クラスは、コンストラクタに渡される設定クラスからマウスオーバー、アウト、ダウン時のスキンを取得し、 ユーザーの操作に自動でレスポンスを返します。 スキンの表示状態遷移にトランジションを伴う場合には Button クラスを継承し、normal, down, over などのメソッドをオーバーライドします。
